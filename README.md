## ‚ö° Quick Start (3 Minutes Setup)

> **üîÑ Fast-changing project?** This MCP server updates frequently! Consider using [Method 2: Editable Install](#-method-2-editable-install-developers--testers) for instant updates.

### Step 1: Install Python Package
```bash
pip install git+https://github.com/bobbymuls/code-review-mcp.git
```

### Step 2: Configure Cursor IDE

1. **Open Cursor Settings** (Ctrl+, or Cmd+,)
2. **Navigate to**: `MCP & Integrations` ‚Üí `New MCP Server`
3. **Find your Python path in cmd**:
   ```bash
   # Find where Python is installed
   which python   # macOS/Linux
   where python   # Windows
   ```

4. **Add this configuration** (using your Python path):

```json
{
  "code-review-mcp": {
    "command": "/full/path/to/your/python",
    "args": ["-m", "code_review_mcp.server"],
    "env": {}
  }
}
```

**Example paths**:
- **Windows**: `"C:\\Python311\\python.exe"` or `"C:\\Users\\YourName\\AppData\\Local\\Programs\\Python\\Python311\\python.exe"`
- **macOS**: `"/usr/local/bin/python3"` or `"/opt/homebrew/bin/python3"`
- **Linux**: `"/usr/bin/python3"` or `"/home/username/.local/bin/python"`

### Step 3: Restart Cursor & Start Coding!

‚úÖ You should see **6 tools enabled** under `code-review-mcp` in MCP settings  
‚úÖ Now you can ask: *"Review this code for security issues"* and get instant analysis!

```python
# Before (vulnerable)

import pickle

def load_data(data):
    return pickle.loads(data)  # ‚ùå Pickle deserialization can be unsafe

# After (secure)
import os
import json
import subprocess

api_key = os.getenv("API_KEY")  # ‚úÖ Environment variable
password = os.getenv("PASSWORD")  # ‚úÖ Environment variable

def execute_command(user_input):
    # ‚úÖ Safe subprocess with shell=False
    result = subprocess.run(["ls", user_input], capture_output=True, text=True)
    return result.stdout

def load_data(data):
    return json.loads(data)  # ‚úÖ Safe JSON deserialization
```

### üåê **API Handling Analysis** (`analyze_api_handling`)
```python
# Before (unreliable API calls)
import requests

def fetch_data(url):
    # ‚ùå No timeout - potential hanging requests
    # ‚ùå No status code checking
    # ‚ùå No error handling
    response = requests.get(url)
    return response.json()

def retry_request(url):
    while True:  # ‚ùå Potential infinite retry loop
        try:
            return requests.get(url).json()
        except:
            time.sleep(5)  # ‚ùå Fixed sleep - no exponential backoff

# After (robust API calls)
import requests
import time
from requests.adapters import HTTPAdapter
from urllib3.util.retry import Retry

def fetch_data(url, timeout=30):
    try:
        # ‚úÖ Proper timeout and status checking
        response = requests.get(url, timeout=timeout)
        response.raise_for_status()  # ‚úÖ Check status codes
        return response.json()
    except requests.exceptions.RequestException as e:
        print(f"API request failed: {e}")
        return None

def retry_request(url, max_retries=3):
    session = requests.Session()
    # ‚úÖ Exponential backoff retry strategy
    retry_strategy = Retry(
        total=max_retries,
        backoff_factor=1,
        status_forcelist=[429, 500, 502, 503, 504],
    )
    adapter = HTTPAdapter(max_retries=retry_strategy)
    session.mount("http://", adapter)
    session.mount("https://", adapter)
    
    try:
        response = session.get(url, timeout=30)
        response.raise_for_status()
        return response.json()
    except requests.exceptions.RequestException as e:
        print(f"Request failed after {max_retries} retries: {e}")
        return None
```

### ü§ñ **LLM Integration Analysis** (`analyze_llm_invoke`)
```python
# Before (unsafe LLM usage)
import openai

def chat_with_ai(user_input):
    # ‚ùå User input directly in prompt - prompt injection risk
    prompt = f"Answer this question: {user_input}"
    
    # ‚ùå Very high max_tokens - cost implications
    # ‚ùå No system message for guidance
    response = openai.chat.completions.create(
        model="gpt-4",
        messages=[{"role": "user", "content": prompt}],
        max_tokens=8000,
        temperature=0.7
    )
    
    # ‚ùå No response validation or stripping
    return response.choices[0].message.content

# After (secure LLM usage)
import openai
import re

def sanitize_input(user_input):
    # ‚úÖ Input validation and sanitization
    cleaned = re.sub(r'[^\w\s\-\.\?\!]', '', user_input)
    return cleaned[:500]  # Limit length

def chat_with_ai(user_input):
    # ‚úÖ Sanitize user input
    safe_input = sanitize_input(user_input)
    
    # ‚úÖ Use system message for better guidance
    # ‚úÖ Reasonable token limits
    response = openai.chat.completions.create(
        model="gpt-4",
        messages=[
            {"role": "system", "content": "You are a helpful assistant. Answer questions concisely and safely."},
            {"role": "user", "content": f"Please answer: {safe_input}"}
        ],
        max_tokens=1000,  # ‚úÖ Reasonable limit
        temperature=0.7
    )
    
    # ‚úÖ Validate and strip response
    result = response.choices[0].message.content.strip()
    return result if result else "Sorry, I couldn't generate a proper response."
```

```
Please conduct a comprehensive code review of [filename] using both automated MCP tools and manual critical analysis. Structure your review using this hybrid methodology:
```
